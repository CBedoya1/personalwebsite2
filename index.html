Hardware Validation Intern, Intel Corporation

July 2014-September 2014

Generated a formal validation model to examine and test the features of a new communication protocol to be used in a large-scale computing system. Incorporated feedback from engineers across sites in order to construct a robust model. Documented the details of the protocol so as to provide other engineering groups with a background on the workings of the system

Software Engineering Intern, Intel Corporation

April 2013-September 2013

Collaborated with cross-disciplinary experts in the yield department, identified areas of inefficiency, and translated them into software solutions. Designed and implemented a monitoring system that allowed yield engineers to detect failures in production tools at an earlier stage in production. Rewrote and restructured existing software to streamline the collecting and reporting of yield data and reduced the task process time by 80%

Electronics Intern, Draper Laboratory

June 2012-August 2012

Engaged in research and development of a smaller and more stable temperature- compensated crystal oscillator (TCXO) to be used in GPS navigation. Developed MATLAB scripts used to program source meters, function generators, and other instruments used for collecting temperature data on TCXO
